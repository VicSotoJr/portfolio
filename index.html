Inventory Management System

Inventory-400.herokuapp.com

Details

Purpose: To create and implement an easy to use inventory management system for a small company that currently uses pen and paper.

Current State: Can view and edit current inventory. Can also add new items and delete old items. Physical scanner can be used in order to scan items instead of manually changing values. Orders can be placed automatically to a third party site.

Future Improvements: Make the system more visually appealing.

Details
